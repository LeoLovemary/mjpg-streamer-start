# mjpg-streamer-start
mjpg-stareamer auto restart program

使用方法：
  1.提取 mjpg-streamer-start.cpp 文件内容
  2.linux环境下<仅支持此项>
    $vim mjpg-streamer-start.cpp
      -->嵌入提取的内容
    $wq!
      -->保存
    $g++ -o mjpg-streamer-start mjpg-streamer-start.cpp
    $mjpg-streamer-start & 
      -->成功运行
